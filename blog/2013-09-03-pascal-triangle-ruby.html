帕斯卡三角形 (Ruby)

Sep 3rd, 2013 | Comments

帕斯卡三角形

def pascal_triangle(n, i = 2, result = [[1], [1,1]])
  if n == 1
    [[1]]
  elsif n == 2
    [[1], [1,1]]
  elsif i == n
    result
  else
    current_row = [1]
    (i-1).times do |n|
      current_row << (result[i-1][n] + result[i-1][n+1])
    end
    current_row << 1

    pascal_triangle(n, i+1, result << current_row)
  end
end

Output

pst = pascal_triangle(7)

puts pst.pretty_inspect

=begin
[[1],
 [1, 1],
 [1, 2, 1],
 [1, 3, 3, 1],
 [1, 4, 6, 4, 1],
 [1, 5, 10, 10, 5, 1],
 [1, 6, 15, 20, 15, 6, 1]]
=> nil
=end

Check

pst.map {|row| row.reduce(:+) == 2 ** (row.length - 1) }
# => [true, true, true, true, true, true, true]
